How do I create a new feature?

Prerequisites

  1. Enable editing mode () in the tool bar.
  2. Select the layer you want to create in. This can normally be done in the layer control. In this show-case, the vector layer is already selected.
    Hint: you can only create new features in vector-layers.

Creating a new feature

  1. Start: Click the RIGHT mouse button on the map, and a menu will appear. At the top of this menu, you will see "Create new feature". Select this option to start.
  2. Create: After the "Create new feature" option has been selected, just start clicking to place new points on the map.
  3. Save: Click the RIGHT mouse button again, and a new menu will appear. Select the option "Save editing".
