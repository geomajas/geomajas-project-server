How do I edit an existing feature?

Prerequisites

  1. Enable editing mode () in the toolbar.
  2. Select the feature you want to edit on the map (left mouse click).
    Hint: by selecting a feature, it will become yellow.

Editing the selected feature

  1. Start: Click the RIGHT mouse button on the map, and a menu will appear. Select "Edit selected feature" to start.

    Hint: as long as the orange edges and points are drawn, the feature is in editing mode. Except for point layers using images; there the image changes.
  2. Drag a point: You can now drag the end-points of the geometry around.
  3. Insert a point (not available for point layers): Click the RIGHT mouse button precisely on one of the orange edges. Then select "Insert point".
  4. Remove a point (not available for point layers): Click the RIGHT mouse button precisely on one of the end-points. Then select "Remove point".
  5. Undo last step: Click the RIGHT mouse button, then select "Undo last operation". This will make your last change undone.
  6. Save: Click the RIGHT mouse button again, and a new menu will appear. Select the option "Save editing".
