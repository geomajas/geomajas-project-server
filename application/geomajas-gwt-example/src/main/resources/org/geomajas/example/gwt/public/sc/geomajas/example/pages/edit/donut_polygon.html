How do I create a donut polygon?

This option is possible only on polygon layers!

Prerequisites

  1. Enable editing mode () in the toolbar.
  2. Select the feature you want to edit on the map (left mouse click).
    Hint: by selecting a feature, it will become yellow.

Editing the selected feature, to create a hole in it

  1. Start: Click the RIGHT mouse button on the map, and a menu will appear. Select "Edit selected feature" to start.

    Hint: as long as the orange edges and points are drawn, the feature is in editing mode. Except for point layers using images; there the image changes.
  2. Create hole: Click the RIGHT mouse button on the feature, and a menu will appear. Select "Add interior ring".
  3. Place points: Click new points inside the feature's border.
  4. Stop inserting: Click the RIGHT mouse button, and click "Add point modus", so it is deselected.
    Hint: by deselecting the "Add point modus", you are now in normal editing mode. This means you can drag points around, etc.
  5. Save: Click the RIGHT mouse button again, and a new menu will appear. Select the option "Save editing".
